# 데이터시각화 3차 시험 - Stremalit 애플리케이션 설명서 
**산업데이터공학과 C373022 장윤서** 

## 1. 기획의도 
본 애플리케이션은 네이버 뉴스 검색 API로 수집한 기사(제목·요약)를 기반으로, 넷플릭스 「K팝 데몬 헌터스」 관련 이슈를 정리하고 팬덤 형성의 핵심 요인을 시각적으로 분석하기 위해 제작했다.  
기사량(언론 노출) 변화, 반복 키워드, 키워드 간 연결 구조를 함께 확인하여 “왜 특정 시점에 관심이 폭발했는지”와 “어떤 담론 축으로 소비되는지”를 파악하는 데 목적이 있다.

## 2. 데이터 전처리 
- 데이터 소스: Naver News API
- 분석 단위: 기사 title+description 텍스트 
- 전처리: HTML 태그 제거, 공백 정리, 날짜·시간 변환(서울 시간 기준), 명사 추출(Okt)

## 3. 주요 기능 및 사용법 

### 3-1. 사이드바 설정(필터/조절) 
- 기간 선택: 분석 구간을 설정한다. 모든 시각화가 선택 기간 기준으로 갱신된다.
- 집계 단위(일별/주별) : 기사량 추이를 촘촘하게(일별) 또는 흐름 중심으로(주별) 확인한다. 
- TopN 키워드: 키워드 분석 범위를 조절한다 
- 네트워크 min_count: 동시출현 네트워크에서 약한 연결을 제거해서 핵심 구조만 확인할 수 있다
- 중심성 TopN: 네트워크에서 연결이 많은 핵심 키워드 표로 확인한다. 
- 포커스 키워드: 특정 키워드가 포함된 기사만 모아 표 및 네트워크 맥락을 좁혀 분석한다.

## 4. 시각화 구성과 해석 

### 4-1. 데이터 요약 지표 
- 선택기간의 기사 수, 분석 일수, 피크 날짜, 피크 기사 수를 제공한다 
- 피크날짜는 관심이 집중된 시점을 빠르게 찾기 위한 기준점으로 활용된다. 

### 4.2. 관심도 추이 - Altair 
- 기간에 따른 기사량 변화를 통해 이슈의 확산 흐름을 확인한다.  
- 본 데이터에서는 12/09 전후에 기사량이 크게 상승하며, 골든글로브 관련 이슈(후보 발표)가 확산의 촉발 요인으로 해석된다.  
- 그 근거 자료는 12/09 기사를 첨부해두었다. 
- 피크 이후 하락이 빠른 경우, 장기 담론이라기보다 이벤트성 보도로 단기 집중된 패턴일 수 있다.

### 4-3 핵심 키워드 Top N - seaborn 
- 기사에서 반복되는 키워드를 상위 순위로 제시한다.  
- 작품 고유 키워드(제목·플랫폼)가 상위에 나타나는 것은 이슈가 명확한 하나의 대상에 집중되어 있음을 의미한다.  
- 동시에 애니메이션/영화/감독/콘텐츠, 글로벌/미국/세계, 골든글로브/후보 같은 키워드가 함께 등장ek면, 팬덤 담론이 “작품성 평가 + 해외 반응/외부 인증” 축으로 확장되는 흐름을 시사한다.

### 4-4 담론 프레임 비중 - Plotly
- 키워드를 ‘시상식/성과’, ‘글로벌/해외’, ‘콘텐츠/제작’, ‘음악/아이돌’ 프레임으로 묶어 기사들이 어떤 관점으로 설명되는지 비중으로 확인한다.  
- 특정 프레임이 높게 나타나면, 해당 기간에 언론이 이 작품을 어떤 서사로 주로 소비했는지(성과 중심/글로벌 반응 중심 등)를 정리할 수 있다.

### 4-5. 워드클라우드 
- 선택 기간에 반복되는 단어를 직관적으로 보여준다.  
- 막대그래프가 순위 비교라면, 워드 클라우드는 현재 담론의 분위기와 단어군을 빠르게 파악하는 역할을 한다. 

### 4-6. 포커스 키워드 기반 기사 확인 테이블 
- 포커스 키워드를 선택하면 해당 키워드가 포함된 기사만 표에 모인다.  

### 4-7. 키워드 동시 출현 네트워크 (NetworkX)
- 같은 기사에서 함께 등장한 키워드 쌍을 연결해 담론 구조를 시각화한다.  
- 노드가 클수록 다양한 키워드와 함께 언급되는 “중심 키워드”로 해석할 수 있다.  
- min_count를 조절하면 복잡한 연결을 정리해 핵심 구조를 보기 쉽게 만들 수 있다.  
- 중심성 Top 키워드는 해당 기간 담론의 허브 역할을 하는 키워드를 요약해준다.
- 추가적으로, 케데헌 대표 색상인 보라색으로 노드 색을 구성하고, 글씨가 잘 보일 수 있게 흰색 글씨로 변경하였다. 

## 5. 종합 결론 
본 대시보드는 기사 데이터 기반으로 팬덤 형성 과정의 단서를 “노출(기사량)–내용(키워드)–구조(네트워크)” 관점에서 정리한다.  
특정 시점(피크)에는 시상식/성과 같은 외부 이벤트가 관심을 급격히 끌어올릴 수 있고, 이후에는 콘텐츠/제작·글로벌 반응·음악/아이돌 요소가 결합되며 담론이 확장되는 흐름을 확인할 수 있다.  
다만 기사량은 팬덤 규모 자체가 아니라 언론 노출 지표이므로, 키워드·기사 맥락·네트워크 구조를 함께 보며 해석하는 방식이 필요하다.
